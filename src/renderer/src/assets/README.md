# 视频资源说明

本目录包含答题反馈动画视频文件：

- `right.mp4` - 答对时的动画视频
- `wrong.mp4` - 答错时的动画视频

## 重要提示

⚠️ **这些视频文件不在 Git 仓库中**（已添加到 `.gitignore`）

### 为什么？

- 视频文件较大，不适合存储在 Git 仓库中
- 避免仓库体积过大
- 便于单独管理和更新视频文件

### 如何添加/更新视频文件？

1. 直接将视频文件放到 `src/renderer/src/assets/` 目录
2. 确保文件名为 `right.mp4` 和 `wrong.mp4`
3. 文件会自动被 Vite 打包到应用中

### 视频文件要求

- **格式**: MP4 (H.264 编码推荐)
- **尺寸**: 建议 1280x720 或更高
- **大小**: 尽量压缩，建议 < 1MB
- **时长**: 建议 2-5 秒

### 构建说明

- 本地开发：确保视频文件存在于 `src/renderer/src/assets/` 目录
- CI/CD 构建：需要在构建前将视频文件放到正确位置，或使用其他方式（如 CDN、外部资源等）
